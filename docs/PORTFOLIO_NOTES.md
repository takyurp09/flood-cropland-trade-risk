# Portfolio Notes

This is a public, limited-scope portfolio repository. It is meant to show research computing ability, not to release the full unpublished paper.

## Included

- Reproducible project structure
- Curated Python scripts from the flood, trade, panel, validation, and projection pipeline
- Google Earth Engine workflow notes
- Data-source documentation
- Selected figures that demonstrate scale and method

## Excluded

- Raw data
- Access tokens and `.env` files
- Local machine paths
- Manuscripts, submission packages, referee-response files, and private notes
- Full result tables or thesis-only outputs
- Large intermediate and final outputs

## Why This Fits a Research Computing Portfolio

The repository demonstrates:

- remote-sensing and Earth observation processing;
- Google Earth Engine and geospatial data workflows;
- agricultural exposure measurement;
- trade-network construction;
- panel-data assembly for applied economics;
- validation using independent food, crop, and vegetation data;
- forward-looking climate-risk projection structure;
- GitHub-ready documentation and version-control hygiene.
