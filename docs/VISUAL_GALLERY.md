# Visual Gallery

This repository includes a small set of public-facing visuals selected to demonstrate workflow scale, remote-sensing logic, and communication style. They are included as portfolio artifacts, not as a full release of manuscript figures.

## Included Figures

| File | What it demonstrates | Public-release role |
|---|---|---|
| `figures/selected/workflow_overview.svg` | End-to-end data flow from satellite flood layers to crop exposure, trade exposure, panel construction, validation, and projection inputs | Shows research design and reproducible pipeline structure |
| `figures/selected/global_flood_trade_exposure_map.png` | Global spatial framing for flood-linked food-trade exposure | Shows the project is built for large-scale geospatial and trade-risk analysis |
| `figures/selected/sensor_crosswalk_diagnostic.png` | Diagnostic comparison across flood products or sensor-derived measures | Shows validation-oriented remote-sensing practice rather than treating one product as automatically correct |

## Why Only Selected Visuals Are Included

The full research project contains additional internal diagnostics, robustness plots, and manuscript tables. Those are intentionally excluded from this public portfolio release because they may contain unpublished results, large intermediate outputs, or thesis/manuscript-specific framing.

## Reviewer Takeaway

The visual set is meant to signal three capabilities relevant to agricultural economics and remote sensing roles:

- translating Earth-observation products into applied economic exposure measures;
- building interpretable geospatial and trade-network summaries;
- documenting validation and quality-control steps in a reproducible workflow.
