# Example Schemas

These CSV files are small real-data extracts from processed intermediate products. They document expected column names and data shapes without releasing raw data, full panels, or final empirical result tables.

- `sample_flood_cropland_exposure.csv`: exporter-crop-year flood cropland exposure extract.
- `sample_flood_trade_exposure.csv`: importer-crop-year flood trade exposure extract.
- `sample_regression_panel_schema.csv`: selected country-year panel fields used for applied-economics analysis.

Values are intentionally limited to a few rows and should be treated as documentation examples, not as a complete replication dataset.
