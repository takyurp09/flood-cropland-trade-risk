# Example Schemas

These CSV files are small illustrative examples. They document expected column names and data shapes without releasing raw data or empirical results.

- `sample_flood_cropland_exposure.csv`: exporter crop exposure structure.
- `sample_flood_trade_exposure.csv`: importer trade exposure structure.
- `sample_regression_panel_schema.csv`: final country-year panel structure.

Values are synthetic and are included only for documentation.
